Connect Airtable

Enter your Airtable PAT and Base ID to get started

Create a PAT at airtable.com/create/tokens with scopes: data.records:read + data.records:write + schema.bases:read

1. Go to airtable.com/create/tokens
2. Click "Create new token", name it "TableTurn"
3. Select scopes: data.records:read, data.records:write, schema.bases:read
4. Copy token (starts with pat...) and paste above
5. Find Base ID in URL: airtable.com/appXXXXXXX/...

🔒 Your PAT is stored only in your browser and never sent to our server.
